Open

Open a Word, Excel, CSV, text or PDF file. Nothing is uploaded.

Review

Penname suggests a pen name for every value it finds. You check each one. This step is never skipped.

Export

Save the pen-named copy and paste it into your AI assistant. The encrypted key file stays with you.

Take the pen names off

Paste the reply back. Penname restores the real values to a file on your computer.

Penname reduces the personal data you share. It does not make data anonymous. No tool finds everything, which is why the review step is required.
