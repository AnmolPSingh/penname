# Document furniture: words a proposal or report Title Cases as headings and
# section labels, which the language model then reads as names, organisations
# or places. One entry per line, lowercase. Comments start with #.
#
# Only consulted for language-model guesses (names, organisations, places,
# groups, dates). Pattern-backed values such as emails, phone numbers, gift
# amounts, fund codes and constituent IDs are never checked against this list.
#
# Trade-off: an organisation genuinely called "Impact" or "Reach" will not be
# picked up automatically. The review screen's "Add something Penname missed"
# button covers that case, and it is far rarer than the heading it would
# otherwise flag on every page.
